TR:
    Özet:
        1- Bu plugin Redmine'daki işlerin sürelerini, kimin oluşturduğunu vb. özelliklerini takip eder.
    Özellikler:
        1- Hangi iş durumunda ne kadar süre durulduğu.
        2- Üzerinde durulan işin kategorisi.
        3- Durum değişikliğinin kim tarafından yapıldığı.
        4- İşte harcanan zaman.
        5- Proje bazlı açıp kapanılabilirlik. (init.rb'de)
    Kurulum:
        1- Plugin dosyasını indirin.
        2- Bu dosyayı önceden indirdiğiniz Redmine projesinin Plugin klasörüne ekleyin.
        3- Redmine uygulamasını yeniden başlatınız.

    Sürüm:
        Redmine: 5.1.4


ENG:
    Overall:
        1- This plugin tracks the duration of jobs in Redmine, who created them, and other related details.
    Features:
        1- How long the work status remains in each status.
        2- The category of the work being held.
        3- Who made the status change.
        4- Time spent on the work.
        5- Project based enabling/disabling (at init.rb).
    How To Setup:
        1- Download the Plugin Folder.
        2- Put the Plugin Folder inside your Redmine project's plugin folder
        3- Restart your Redmine app

    Version:
        Redmine: 5.1.4
